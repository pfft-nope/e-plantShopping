# e-plantShopping – Paradise Nursery

e-plantShopping is a React-based shopping application for an online plant store called **Paradise Nursery**. The application allows users to browse a selection of houseplants, add items to a shopping cart, manage quantities, and view the total cost before checkout.

## Project Features
- Landing page with a background image and company introduction
- Product listing page displaying multiple houseplants grouped into categories
- Add to Cart functionality with disabled state after adding
- Shopping cart page with:
  - Increase and decrease item quantity
  - Remove items from the cart
  - Display of total items and total cost
- Global state management using Redux
- Navigation bar with live cart item count
- Deployed using GitHub Pages

## Technologies Used
- React
- Redux Toolkit
- React Router
- JavaScript (ES6)
- CSS
- GitHub Pages
