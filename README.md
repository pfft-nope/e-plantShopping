# e-plantShopping – Paradise Nursery

e-plantShopping is a React-based online shopping application for a plant store called Paradise Nursery. The application allows users to browse a variety of houseplants, add them to a shopping cart, manage quantities, and view the total cost before checkout.

## Features
- Landing page with background image and company introduction
- Product listing page with plants grouped into categories
- Add to Cart functionality with button disable after adding
- Shopping cart page with:
  - Increase and decrease item quantity
  - Remove items from the cart
  - Display of total plant count and total cost
- Redux for global state management
- Navigation bar with live cart item count
- Deployed using GitHub Pages

## Technologies Used
- React
- Redux Toolkit
- React Router
- JavaScript
- CSS
- GitHub Pages
